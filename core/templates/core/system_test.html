# 测试一：发送非攻击流量

本测试将在后台执行以下命令，其中test1.pcap文件包含4条非攻击流量：
sudo python dispatcher.py ./pcap/test1.pcap
测试已完成。
可在防火墙日志中查看防火墙动作，在流量统计中查看数据包信息，在节点状态中查看交换机流表项。

# 测试二：发送攻击流量

本测试将在后台执行以下命令，其中test2.pcap文件包含8条攻击流量：
sudo python dispatcher.py ./pcap/test2-dos.pcap
测试已完成。
可在防火墙日志中查看防火墙动作，在入侵检测日志中查看入侵检测器动作，在流量统计中查看数据包信息，在节点状态中查看交换机流表项。
